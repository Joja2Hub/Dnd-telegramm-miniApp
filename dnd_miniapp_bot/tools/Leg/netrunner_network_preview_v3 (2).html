Броски по действию

Поля d20 больше не висят постоянно. Нажатие “Начать взлом” или “Атака” открывает отдельную панель ввода значения.

Бой снизу

Когда нетраннер попадает на живой скрипт, снизу появляется боевой док: слева игрок, справа скрипт. Активная сторона подсвечивается.

Туман по уровню

Неизвестные блоки не отображаются карточками. Видны только туннели. Мастер может открыть структуру уровней без подробностей.

SUCCESS / ERROR

Взлом показывает глич-загрузку, а затем явный зелёный SUCCESS или красный ERROR.

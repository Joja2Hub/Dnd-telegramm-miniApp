# Summary по кастомизации v48

## Что добавлено

### Новая база эффектов
В проект добавлена новая библиотека эффектов из демонстрации v13: fantasy, dark, sci-fi и cyberpunk эффекты.

Ключевые новые эффекты:
- Прыжок в сверхсвет — теперь эффект идёт по периметру аватарки.
- Фазовый облёт — объект облетает аватарку по кругу.
- Меридиональный облёт — вертикальная орбита, визуально заходит за аватарку и возвращается на передний план.
- Глитч-контур — глитч по периметру.
- Трассирующий след — летящий объект с трейлом.
- Инфернальное пламя — языки огня, дым и угли.
- Орбитальное сканирование — sci-fi сканирующие линии.
- Неоновый дождь, Чёрный дождь, Снежный шквал, Чёрный лёд, Кибер-призрак и другие.

## Оптимизация анимаций

Эффекты больше не анимируются постоянно.

Анимация запускается только:
- при наведении мышкой на аватарку / превью эффекта;
- при нажатии / тапе по аватарке или превью.

Повторное нажатие снимает фиксацию эффекта. Одновременно фиксируется только один эффект, чтобы страница не лагала.
